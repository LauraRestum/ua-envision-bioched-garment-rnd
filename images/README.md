## Image Assets Required

Place the following files in this directory before deploying `experience.html`.

| Filename | Description | Used in |
|---|---|---|
| `hero-jacket.jpg` | Clean front-facing jacket, white/removed background | S1 Hero, S6 CTA |
| `material-layers.jpg` | Layered textile cross-section macro | S2 Material System |
| `construction.jpg` | Open jacket showing interior construction | S3 Construction |
| `zones-macro.jpg` | Shoulder macro showing material zone junction | S4 Performance Zones (background) |
| `cutter.jpg` | Industrial CNC fabric cutting machine | S5 Infrastructure |
| `bg-texture.jpg` | Dark navy woven textile texture | Global background (all slides) |

All images should be optimized for web (JPEG, 80–90% quality). Hero jacket works best with a transparent or removed background (PNG also acceptable for `hero-jacket`).
