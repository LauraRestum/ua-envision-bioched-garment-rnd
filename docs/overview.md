# Project Overview

## Purpose

Develop a clear, sales-led briefing that presents a believable and fundable pathway from material innovation to deployable garment systems.

This is not a marketing document. It must feel real, specific, and executable to defense R&D stakeholders and commercial partners.

## Core System Model

Three parties. Each owns a distinct, non-overlapping function:

**Under Armour** → material innovation  
**Envision** → rapid prototyping + garment integration + pilot manufacturing  
**Government (DEVCOM)** → funding + evaluation via OTA

The model works because no single party can do all three. UA does not own the prototype-to-deployment pipeline. Envision does not develop advanced materials. The government needs a defined use case and a viable team to fund.

## Use Case

Next-generation bio/chemical protective garment.

The anchor problem: bio/chem protection currently requires tradeoffs between protection level, mobility, and thermal load. No current system resolves all three without significant iteration time and manufacturing gaps.

## Goal

Compress the timeline from concept to deployable system by combining:
- UA's material R&D depth
- Envision's speed and manufacturing capability
- Government funding and evaluation infrastructure

## Output

A controlled, linear interactive briefing structured for a live meeting with Under Armour and defense R&D stakeholders.
